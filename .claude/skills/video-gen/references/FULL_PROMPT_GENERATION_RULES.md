# 完整提示词生成规则

## 提示词结构

`complete_prompt` 只包含动态内容（不含固定前缀）：

```
[场景布局] [镜头1描述] [镜头2描述] ... [音效提示词]
```

每个 Clip 同时生成 v1（`complete_prompt`）和 v2（`complete_prompt_v2`）两个版本。v2 基于相同核心信息独立重新生成，在景别、运镜、构图、光影等方面产生差异化表达。

---

## 模型分支说明

提示词规则根据目标视频模型分为两个版本，通过 `config.yaml` 中 `video_model.active_model` 控制：

| 字段 | 可灵 Kling Omni (`kling_omni`) | 即梦 SD2.0 (`seedance2`) |
|------|-------------------------------|--------------------------|
| 角色定义 | 精通可灵视频Omni模型的视频生成提示词专家 | 精通即梦SD2.0模型的视频生成提示词专家（SD2.0对动作细节极为敏感） |
| **构图规则** | 三分法/中心构图/对角线构图/引导线构图/过肩构图等经典电影构图 | **用大白话**描述人物位置：人物居中/人物偏左/人物位于画面左侧/两人分列画面两侧/说话者在背景听者后脑占前景等 |
| **焦距/镜头感** | 具体焦段mm数值：超广角15-20mm / 广角24-35mm / 标准40-60mm / 中长焦70-105mm / 长焦135-200mm | **描述性词汇**：广角镜头/长焦镜头/长焦压缩感/浅景深/深景深/背景虚化，**不使用mm数值** |
| **动作动态标题** | 动作动态（必须详细描述） | 动作动态（必须详细描述——**SD2.0最吃细节，必须拆解为微动作链**） |
| v2 构图示例 | 如 v1 三分法构图，v2 可换对角线构图或引导线构图 | 如 v1 人物居中，v2 可换人物偏左大量留白，或说话者在背景听者后脑占前景 |

其余所有规则（运镜、景别、光影、对话、连续性、字数等）**两个模型完全相同**。

## 完整提示词必须包含的元素

### 1. 时间标记（必须）
- 格式：`0–3s, 3–7s, 7–10s`（使用 en dash `–`）
- 每个 shot 对应一个时间段
- 时间从 0 开始，连续不间断

### 2. 镜头语言（必须）

包含以下元素：

**a) 景别（Shot Size）**
- 大远景 Extreme Wide Shot
- 远景 Long Shot
- 全景 Full Shot
- 中景 Medium Shot
- 中近景 Medium Close-Up
- 近景 Close Shot
- 特写 Close-Up
- 大特写 Extreme Close-Up
- 俯拍 High Angle / Bird's Eye View
- 仰拍 Low Angle

**b) 运镜（Camera Movement）**
- 固定镜头 Static Shot
- 推镜 Dolly In / Push In（缓慢推镜）
- 拉镜 Pull Back / Dolly Out
- 摇镜 Pan
- 跟拍 Follow Shot（平稳跟拍）
- 环绕 Orbit（轻微环绕）
- 升降 Crane
- 手持 Handheld
- 稳定器 Gimbal
- 一镜到底 One Take
- **稳定描述词（必须附加）**：无抖动、丝滑流畅、平稳、匀速

**c) 焦距/镜头感——模型分支**

> **可灵 Kling Omni**：使用具体焦段mm数值
> - 超广角 15-20mm（大透视/畸变/压迫感）
> - 广角 24-35mm
> - 标准 40-60mm
> - 中长焦 70-105mm
> - 长焦 135-200mm（透视压缩/空间拉平）

> **即梦 SD2.0**：使用描述性词汇，**不使用mm数值**
> - 广角镜头
> - 长焦镜头
> - 长焦压缩感
> - 浅景深
> - 深景深
> - 背景虚化

**d) 景深（Depth of Field）**
- 浅景深 Shallow depth of field
- 深景深 Deep depth of field
- 背景虚化 Background bokeh

**e) 构图（Composition）——模型分支**

> **可灵 Kling Omni**：三分法 / 中心构图 / 对角线构图 / 引导线构图 / 过肩构图等经典电影构图

> **即梦 SD2.0**：用大白话描述人物在画面中的位置关系，如：
> - 人物居中
> - 人物偏左 / 人物位于画面左侧
> - 人物偏右 / 人物位于画面右侧
> - 人物位于画面左下角 / 右上角（制造斜线张力）
> - 两人分列画面左右两侧（对话/对峙场景）
> - 说话者在背景，听者后脑占前景（过肩视角）
> - 人物在画面一角，大片留白（孤独/压抑感）
> - 人物被前景元素框住（门框/窗框构图，强调处境）

**f) 画面张力技法（Visual Tension）**
- **大透视 Extreme Perspective**：超广角近摄主体，夸张透视拉伸空间，制造压迫感或宏大感（如近摄仰拍建筑使线条急剧汇聚）
- **前景压迫 Foreground Oppression**：大体量前景元素占据画面下方或边缘，挤压视觉空间，增强紧张感与纵深
- **汇聚线透视 Converging Lines**：走廊/道路/建筑列柱等线条向消失点汇聚，引导视线同时制造纵深张力
- **广角畸变 Wide Angle Distortion**：超广角极近距离拍摄人物/物体，产生夸张形变与视觉冲击
- **景深张力 Depth Tension**：极浅景深 + 强虚化，或前景锐利/背景模糊的强烈层次对比，制造视觉紧绷感
- **透视压缩 Telephoto Compression**：长焦压缩前后景距离，让画面元素显得密集拥挤，增强压迫氛围

### 3. 光影描述（必须）
- 光源方向：侧光、顶光、逆光、正面光
- 光线质量：柔和、刺眼、温暖、冷峻
- 明暗对比：高调、低调、强对比
- 氛围：金色光晕、蓝色冷光、暖黄色调

示例：
- "午后暖光透过丝绸帷幕洒入，柔和金色光晕"
- "刺眼的午后阳光从窗户射入，形成强烈明暗对比"

### 4. 场景描述（必须）
- 内/外景标注
- 场景使用 `{loc_xxx}` ID 格式引用
- 环境细节：装饰、家具、氛围

示例：
- "{loc_001}内景，华丽的红木家具，丝绸帷幕"

### 5. 人物描述（必须）

**a) 位置和姿态**
- 在场景中的位置
- 身体姿态
- 与其他角色的空间关系

**b) 表情和眼神**
- 面部表情的细节
- 眼神的方向和情绪
- 微表情的变化

**c) 动作细节——模型分支**

> **即梦 SD2.0（重要额外强调）**：动作动态标题附注"SD2.0最吃细节，必须拆解为微动作链"

- **拒绝抽象动词**：禁止使用"跳舞/做饭/走路/战斗"等泛化词汇，必须拆解为微动作链
  - ❌ 错误："女生做饭" → ✅ 正确："女生优雅翻炒，无油烟，轻撒调料，动作连贯自然"
  - ❌ 错误："他战斗" → ✅ 正确："他右脚蹬地前踏，横剑格挡，随即推掌反击，动作流畅不僵硬"
- **必须加稳定词**：每个动作描述需包含流畅感，如"缓慢"、"平稳"、"流畅"、"不僵硬"、"无卡顿"、"动作自然连贯"
- 具体的动作描述（起始 → 过程 → 结束）
- 动作的速度和力度
- 肢体语言

**d) ID 标注**
- 主要角色使用 `{act_xxx}` 格式，如 `{act_001}`、`{act_002}`
- 次要角色/配角/群众等不使用 ID，直接描述，如"几名守卫"、"远处的行人"

### 5.5 敏感词替换（硬性要求）

以下词汇会触发视频平台内容审核，生成时必须替换，不得直接出现在提示词中：

| 原词 / 原词组 | 替换为 |
|-------------|--------|
| 鲜血、血迹、血液、血水、血流、血泊、血腥 | 红色液体、红色痕迹、红色印记 |
| 血溅、血雾、血染 | 红色飞溅、红色弥漫、染红 |
| 死亡、死去、死了、已死、死状 | 消逝、倒下、失去意识、气绝 |
| 杀死、杀掉、斩杀、斩首、砍杀 | 击败、制服、压制、击倒 |
| 杀人、杀敌 | 击倒对手、击败敌人 |
| 尸体、尸骸、尸首、遗体 | 倒地的身影、失去意识的身躯 |
| 割喉、割颈、割腕 | 划过颈部、触碰手腕 |
| 爆头、穿脑 | 强力击中头部 |
| 断肢、残肢、断手、断臂、断腿 | 受创的肢体、倒地的身影 |
| 内脏、肠子、骨髓 | 生命力、内力 |
| 虐待、折磨、凌迟 | 压制、制服 |
| 自杀、轻生、寻死 | 放弃抵抗、失去意志 |
| 爆炸（写实场景）| 强烈冲击、气浪席卷 |

**替换原则**：
- 保留动作的视觉效果和情绪张力，只替换触发审核的具体词汇
- 替换后描述须自然通顺，不影响画面的戏剧性
- 仙侠/奇幻场景中带修饰前缀的词（"仙血""灵血""魔气"等）可保留；"鲜血""血腥"等写实词必须替换
- 直接用替代词描述，无需标注已替换

### 6. 道具描述（如有）
- 有 ID 映射的道具使用 `{prp_xxx}` 格式，如 `{prp_001}`
- 无 ID 映射的道具直接描述名称
- 描述道具的状态和使用方式

### 7. 对话（如有）
- 格式：`{角色ID}（情绪）：对话内容`
- 放在镜头描述的**末尾**
- 保留中文原文，标注语气和情绪

示例：
- `{act_002}（提醒语气）：师姐，你可千万不要修复师兄的丹田啊。`

### 8. 内心独白处理（如有）
- 格式：`{角色ID}内心OS：内容`
- 放在对话之后（或无对话时在描述末尾）
- **重要**：不直接写出内心独白文字，而是通过视觉细节（表情、眼神、微动作）暗示内心活动

错误做法：
```
{act_001}想："百年的婚约和相伴……"
```

正确做法：
```
{act_001}眼神空洞，眼角缓缓凝聚出一滴泪水。他的目光扫过周围堆满的奢华礼物，此刻看来像是残酷的嘲讽。
```

## 对话镜头规则

### 对话切镜（重要）

对话超过 **10 个字**时，**必须体现切镜效果**，禁止用单一固定镜头拍完整段对话：
- 使用"切至"、"镜头切换到"等明确切镜词汇标记镜头变化
- **多人对话**：使用正反打（over-the-shoulder）交替展示说话者
- **单人独白/长台词**：通过推拉镜头制造景别变化（如从中景缓慢推入至近景特写）

示例：
```
中景，{act_001}面对{act_002}开口说话。切至近景，{act_001}表情凝重。切至过肩镜头，{act_002}微微点头回应。
```

### 对话时长计算

- 每 10 个字约需 2 秒
- `建议时长 = max(基础动作时长, 对话时长) + 0.5秒缓冲`
- 基础动作时长：简单动作 2-3 秒，复杂动作 3-4 秒
- 时长范围：最少 2 秒，最多 4 秒
- 对话过程中应有切镜变化，但整体时长要覆盖完整对话

## 生成流程

### 步骤1：分析剧本内容
- 提取动作描述
- 提取对话/内心独白
- 理解情绪和氛围

### 步骤2：设计镜头语言
- 根据情绪选择景别（亲密→近景，宏大→远景）
- 根据节奏选择运镜（紧张→快速，舒缓→缓慢）
- 根据重点选择焦距/镜头感（人物→长焦/长焦镜头，环境→广角/广角镜头）——按 active_model 使用对应词汇

### 步骤3：扩展视觉细节
- 添加环境描述
- 细化人物表情和动作
- 添加光影氛围

### 步骤4：添加 ID 标注
- 主要角色 → `{act_xxx}`
- 场景 → `{loc_xxx}`
- 道具 → `{prp_xxx}`

### 步骤5：整合时间线
- 按 shot 分段
- 添加时间标记（en dash `–`）
- 确保连贯性

## 字数限制

### 总字数

**complete_prompt 总字数不超过 750 字**，包含：
- 场景布局（20-40字）
- 所有镜头描述（约680字）
- 音效提示词（25字）

### 单个镜头字数分配

- **1个镜头**：不超过 500 字
- **2个镜头**：每个不超过 250 字
- **3个镜头**：每个不超过 160 字
- **4个镜头**：每个不超过 120 字
- **5个及以上**：每个不超过 100 字

AI 生成时根据镜头数量自动控制字数，超出限制会自动截断（保留完整镜头，不添加省略号）。

### 精简优先级

- **必须保留**：景别、运镜、动作动态、人物位置、对话内容
- **可以精简**：光影细节、氛围描述、背景细节
- **避免重复**：不要在每个镜头中重复描述相同的场景元素

生成提示词后，检查以下项目：

- [ ] 包含时间标记（0–3s, 3–7s 等）
- [ ] 包含景别描述
- [ ] 包含运镜描述
- [ ] 包含焦距/景深描述
- [ ] 包含光影描述
- [ ] 包含场景描述
- [ ] 包含人物表情和动作细节
- [ ] 主要角色使用 `{act_xxx}` ID
- [ ] 场景使用 `{loc_xxx}` ID
- [ ] 有 ID 的道具使用 `{prp_xxx}` ID
- [ ] 对话格式正确（`{角色ID}（情绪）：内容`）
- [ ] 内心独白转化为视觉描述
- [ ] 对话超过10字有切镜效果
- [ ] 单个镜头字数未超限
- [ ] complete_prompt 总字数 ≤ 750 字
- [ ] 构图规则与当前 active_model 匹配（Kling→经典构图术语，JiMeng→大白话位置描述）
- [ ] 焦距/镜头感与当前 active_model 匹配（Kling→mm数值，JiMeng→广角镜头/长焦镜头等描述性词汇）

## 示例对比

### 简化版（不合格）
```
{act_002}凑到{act_003}耳边，声音压得很低 → 眼神警惕地瞥向{act_001} {act_002}（提醒）：师姐，你可千万不要修复师兄的丹田啊。
```

缺少：时间标记、镜头语言、光影、环境细节

### 完整版（合格）
```
0–3s，85mm中长焦中近景镜头缓缓推入，浅景深。{loc_001}内景，午后暖光透过丝绸帷幕洒入，柔和金色光晕。{act_002}凑近{act_003}耳边，面容紧张而急切，声音刻意压低成耳语，眼神不安地瞥向背景中的{act_001}。{act_003}表情平静但眼中闪过一丝算计。虚化的背景中，{act_001}坐在{prp_001}上一动不动，似乎毫无察觉。{act_002}（提醒语气）：师姐，你可千万不要修复师兄的丹田啊。 3–7s，从{act_003}视角的过肩镜头，标准镜头，中景深。{act_002}的脸部充满画面，眼神强烈而恳求，眉头微蹙。他的手微妙地朝{act_001}方向示意。暖光照亮他额头上的忧虑纹路。背景显示寝宫华丽内景，屏风和丝绸帷幕柔焦虚化。
```

包含：所有必需元素 ✓
