# Subtitle Styles (`assets/styles.json`)

| Preset | Scenario | Features |
|--------|----------|----------|
| horizontal | Landscape video (auto) | Outlined, font size = height * 2.8% |
| horizontal_clean | Landscape video | No outline, same as above |
| vertical | Portrait video (auto) | Outlined, font size = height * 2.2% |
| vertical_clean | Portrait video | No outline, same as above |

Font size, outline width, and bottom margin are all proportional coefficients * video height, auto-adapting to any resolution.
