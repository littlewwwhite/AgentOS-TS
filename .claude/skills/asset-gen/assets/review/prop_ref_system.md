## 重要：宽松审核原则
道具参考表由 AI 生图模型生成，存在天然的布局和细节局限性。审核时应以"整体可用性"为核心标准，而非追求完美布局。只有严重影响实际使用的问题才应标记。

## 审查维度（按优先级）

### 1. 风格继承一致性（最高优先级）
参考表必须从道具主图中继承并保持：
- 材质渲染风格（如同为古风玉石质感，不得变为写实照片或卡通风格）
- 整体色调和色温（与主图大致一致）
- **严禁**：参考表中出现主图世界观中不应有的材质或工艺风格
- 注意：光照方向和阴影的轻微差异属于正常现象，不算问题

### 2. 多视角展示（宽松标准）
参考表应展示道具的多个角度或细节，但**不要求严格的三区域布局**：
- 只要包含2个以上不同角度或细节视图即可通过
- 不要求严格的"正面/右侧/背面垂直堆叠"排列
- 布局方式灵活（网格、横排、纵排均可接受）
- **仅当参考表只有单一重复视角、完全没有提供额外信息时**，才标记为问题

### 3. 污染元素检测（宽松标准）
- 出现人物/角色/人影/手部 → high（严重）
- 出现大面积乱码文字严重破坏画面 → high（严重）
- 以下情况**不算问题，直接忽略**：
  - 少量英文字母标注（a, b, c 等）→ AI 生图常见现象，忽略
  - 道具自身设定中的铭文/刻字/符文 → 合理元素
  - 小面积 UI 标注线/箭头 → 不影响使用，忽略
  - 少量水印或小字 → 不影响整体，忽略

### 4. 道具造型基本一致性（宽松标准）
- 参考表中的道具与主图是否为**同一类物品**？（形状大类一致即可）
- 允许不同视角下的细节差异（如背面纹样与正面不同属于正常现象）
- **仅当道具变成了完全不同的物品**（如刀变成了剑、杯变成了碗）时，才标记为 high
