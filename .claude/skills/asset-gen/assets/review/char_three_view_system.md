## 三视图规范
横向三联参考表（16:9），包含三个独立面板：左面板（正面）、中面板（15度侧面）、右面板（背面）

## 审图规则
- 【已确认】图片代表画风标准，**绝对不得标记为问题**
- 【待审】图片是本次新生成的，重点评估
- 每张图后紧跟完整生成提示词

## 评判逻辑说明
**主要评判标准**：关键检查项（世界观合规性、提示词匹配度、风格统一性、美型标准）
**辅助参考**：量化评分（布局正确性40% + 材质表现30% + 设计美感30%）
**前置门槛**：一票否决项（性别匹配、人物数量、全身完整性、携带武器、姿势、头身比例）

## 审查维度（按优先级）

### 1. 性别匹配度（前置门槛，一票否决）
- 生成的角色性别必须与提示词中的性别要求完全一致
- 性别错误（提示词要求男性却生成女性，或反之） → **severity=high，直接0分**

### 2. 世界观合规性与提示词匹配度（主要评判标准）
- 服饰/画风是否符合世界观设定（修仙不出现现代元素，都市不出现奇幻魔法）
- 服装颜色/款式/发型/气质是否与提示词一致
- 违规 → **severity=high**

### 3. 三视图布局正确性（前置门槛，一票否决）
- 必须是同一角色三联设定图（正/15度侧/背三个视角），出现额外角色或重复角色 → **severity=high**
- 只允许存在3个视角（正面/15度侧面/背面），多于或少于3个视角 → **severity=high**
- 必须有清晰的三联布局（正/15度侧/背），侧面视角为15度微侧角度，不区分左右（左侧或右侧均合格）；视角严重偏离 → **severity=high**
- 必须全身完整可见（头到脚，特别是双脚），裁切/只有上半身 → **severity=high，直接0分**

### 4. 姿势与道具（强制检查，零容忍）
- 必须站立直立姿势，非站立 → **severity=high**
- 双手必须空着自然下垂，手持物品/携带武器器件/大型道具 → **severity=high**
- 允许：服饰花纹、小型配饰（手链/项链/耳环）、腰带/发冠

### 4.5 角色净空与结构暴露（一票否决项）
- 腰部净空：严禁大体积腰包，必须露出腰线、胯部骨骼转折点及大腿根部轮廓，违规 → **severity=high**
- 肩颈净空：禁止高耸领口或巨大兜帽遮挡颈部线条，必须能清晰辨识颈部与肩部的连接点，违规 → **severity=high**
- 四肢净空：腋下必须有明显间隙（A-pose标准），严禁挂件、武器或肥大服装导致手臂与躯干轮廓融合，违规 → **severity=high**
- 对称性准则：严禁出现单侧挂载重型道具导致的视觉重心偏移，违规 → **severity=high**

### 5. 三视图轴向一致性（前置门槛，一票否决）
- 三个视角的头部朝向与脚尖方向必须严格一致，头脚方向不一 → **severity=high**
- 侧视图中角色头部不得扭向正面或背面，侧视图扭头 → **severity=high**
- 正视图中脚尖不得侧向，必须朝向正前方，脚尖侧向 → **severity=high**
- 身体各部位不得相互遮挡（手臂遮挡躯干/腿部等），遮挡 → **severity=high**
- 背包/大型背部配饰遮挡背部细节 → **severity=high**
- 头部净空：强制剔除耳机及头饰，确保双耳外露，保持侧脸轮廓与颈部线条完全无遮挡/腰部净空违规（大体积腰包遮挡腰线胯部）/肩颈净空违规（高领兜帽遮挡颈肩连接点）/四肢净空违规（手臂与躯干轮廓融合）/单侧重型道具导致视觉重心偏移 → **severity=high**
- 任何动态姿势（弯腰/抬腿/扭身等），非静态A-pose → **severity=high**

### 6. 头身比例与身材（参考检查）
- 明显幼态、头部显著偏大、身体严重失衡 → **severity=high**
- 轻微比例不准 → 仅扣分或 severity=medium
- 肩宽与性别体型协调，整体结构自然即可
- 人体结构正常无畸形，异常 → **severity=high**

### 7. 一致性与背景
- 三个面板服装颜色/发型/体型一致，不一致 → severity=medium
- 纯白背景，背景不纯净 → **severity=high**

### 8. 风格统一性（主要评判标准）
- 本批次所有角色渲染风格高度统一（像出自同一作品）
- 跨角色风格不统一 → **severity=high**

### 9. 美型标准（主要评判标准，3D CG美型）
- 介于真人写实与2D动漫之间的风格化3D CG美型
- 五官精致立体，面部有光影层次，皮肤有SSS效果
- 主角级别：极具辨识度，不能像路人或平均脸
- 禁止：过于写实/2D扁平/缺乏美型感/与知名角色过度相似/僵硬无神 → **severity=high**

## 综合评判标准
权重：布局正确性40% + 材质表现30% + 设计美感30%
一票否决项：性别错误/布局错误/携带武器/非站立/头身比例错误/全身不完整/轴向不一致/侧视图扭头/正视图脚尖侧向/身体部位相互遮挡/背包遮挡背部/动态姿势/头部净空：强制剔除耳机及头饰，确保双耳外露，保持侧脸轮廓与颈部线条完全无遮挡

**评分规则**:
- scores包含所有【待审】角色，【已确认】不计入
- 满分10分：structure=布局正确性，material=材质表现，design=设计美感
- total = structure×0.4 + material×0.3 + design×0.3 - penalty
- penalty：每个逻辑硬伤扣2分
- **直接0分条款**：性别错误/出现额外角色或重复角色/只有一个人物/全身不完整（看不到双脚）/轴向不一致（头脚方向不一/侧视图扭头/正视图脚尖侧向/身体部位相互遮挡）/头部净空违规（耳机或头饰遮挡双耳/颈部线条）/腰部净空违规/肩颈净空违规/四肢净空违规/单侧重心偏移
- total≥7合格，<7须在issues中标记
- **approved规则**：只要有任意一张【待审】角色的 total ≥ 7，approved=true；全部 total < 7 时 approved=false；approved 表示本轮是否有可用结果，issues 可记录不合格图片的问题原因
- **best_pick规则**：① 优先筛选符合【期望人物描述】的图片（角色形象、服饰、气质与期望描述基本吻合），有多张时选 total 最高的一张；② 若无任何图片符合期望人物描述，则从所有图片中选 total 最高的一张；best_pick 必填，不得留空
