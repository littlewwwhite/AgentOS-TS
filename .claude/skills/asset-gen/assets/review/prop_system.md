## 审查维度（仅3项，按优先级）

### 1. 设定匹配（4分）
综合检查道具是否符合世界观和提示词描述：
- 材质/工艺是否符合世界观？（如仙侠不该有塑料感，现代不该有古代文物感）
- 道具类型/形状/颜色/材质是否与提示词一致？
- 有无"提示词说A，图片生成了B"的情况？
- **现代道具大众化设计检查**（仅适用于都市/现代/校园/青春等当代世界观）：
  - 道具外观是否符合大众消费品的设计语言？应像真实商品而非概念设计
  - 手机/电脑/家电等电子产品：外观应符合主流品牌的通用设计风格（圆角矩形、简洁线条），不应出现过度科幻/夸张造型
  - 日用品/文具/家具：造型应朴实实用，符合日常生活中常见的款式，不应有奇幻装饰或不合理的艺术化变形
  - 交通工具/工具类：应符合现实中量产产品的外观规范，不应像手工DIY或实验室原型
  - 道具造型过于概念化/科幻化/奇幻化，不像真实生活中能买到的东西 → **severity=high**
- 不匹配 → **severity=high**

### 2. 结构正确性（4分）
综合检查道具的物理结构是否合理：
- **完整性**：道具是否完整可见、未被裁切？
- **物理合理性与功能逻辑**：朝向/开口/握持部位是否符合实际用途？**核心功能结构（如光学镜筒、枪管、刀剑刃线等）是否存在反常理的弯曲、折断或严重错位（如光轴偏离90度、枪管扭曲），导致道具彻底失去现实使用可能？** 重心与平衡感是否可信？
- **比例与连接**：各部件之间大小比例是否协调？多部件衔接是否自然（无断裂/悬浮/不合理融合）？对称道具是否均衡？
- **AI崩坏检测**：有无多余突起物、部件融合变形、悬浮碎片、重复纹理异常、拓扑错误、**违反物理常识的部件强行拼凑**？
- **⚠️ AI生成细节偏差容忍规则**：子部件的精确位置偏差（如凹槽在四角vs四边中间、按钮左右互换、纹样位置偏移等）属于AI出图的天然局限，只要道具整体可辨识且功能逻辑正确，此类偏差 → **severity=medium**，不视为结构错误
- 结构问题 → **severity=high**
- **⚠️ AI生成结构错误封顶规则**：若检测到任何 AI 崩坏问题（**严重功能错位/轴向断裂**、多余突起、融合变形、悬浮碎片、拓扑错误、**部件强行拼凑**等），该图片总分**不得超过6分**，且 structure 分项**不得超过1分**

### 3. 画面质量（2分）
综合检查渲染质量和画面干净度：
- 材质渲染是否精细可信？有无噪点/模糊？
- 背景是否干净（纯白）？道具是否居中？
- **⚠️ 摄影器材穿帮检测（一票否决）**：画面中出现柔光箱、灯架、反光板、摄影棚支架等摄影器材穿帮，严重破坏纯净背景要求 → **severity=high**，该图片总分**直接归零**
- 文字处理：提示词要求的铭文/刻字只要不崩坏就保留；无意义乱码或严重崩坏的文字 → severity=high
- 画面质量差 → **severity=medium**

### 4. 中国国情合规（一票否决）
- 道具不得出现任何违反中国国情的元素，包括但不限于：国旗国徽的不当使用、涉政敏感符号、宗教极端符号、分裂国家相关标识、暴力血腥毒品相关器具的美化呈现、赌博用具、色情暗示元素等
- 出现违规元素 → **severity=high**，该图片总分**直接归零**
