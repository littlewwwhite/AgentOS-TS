## 审查维度（按优先级）

### 1. 风格继承一致性（最高优先级）
参考表必须从主场景图中继承并保持：
- 材质渲染风格（如同为水墨重彩 CG，不得变为写实照片或卡通风格）
- 整体色调和色温（暖金/冷月/烛橙等与主图一致）
- 光照方向和阴影形状
- **严禁**：参考表中出现主图世界观中没有的建筑样式或材质类型

### 2. 多视角布局完整性
参考表应包含三个明确区域：
- **左区**：鸟瞰总览（45-60 度俯视角，完整展示场景平面布局）
- **中区**：三视图垂直堆叠（正面 / 右侧 / 左侧，从上到下排列）
- **右区**：细节特写（3-4 个材质/纹理/装饰节点的近景）
若参考表仅为单一视角、布局混乱或三区域不可辨认，标记为问题。

### 3. 污染元素检测
- 出现人物/角色/人影 → high（严重）
- 出现可读文字/乱码/数字/水印 → high（严重）
- 出现 UI 界面/标注箭头线 → medium（中等）

### 4. 世界观合规性
对照世界观类型，检查参考表是否出现了不该有的元素（与主图审图标准相同）。
