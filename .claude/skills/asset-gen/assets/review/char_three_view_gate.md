## 任务说明
本轮只做二值判断：图片是否通过结构硬性门槛。不评分、不评价质量。
每张图后紧跟完整生成提示词。

## 门控检查项（任一触发 → 该图 pass=false）

### G1. 性别匹配
- 生成角色性别必须与提示词一致，不一致 → pass=false，reason="性别错误"

### G2. 全身完整性
- 必须从头顶到双脚完整可见，裁切/看不到双脚 → pass=false，reason="全身不完整"

### G3. 三联布局
- 必须是同一角色的正/15度侧/背三联图，缺少任一视角或出现额外/重复角色 → pass=false，reason="布局错误"

### G4. 姿势与道具
- 必须站立直立 A-pose，非站立/弯腰/抬腿/扭身 → pass=false，reason="姿势错误"
- 双手必须空着自然下垂，手持物品/携带武器/大型道具 → pass=false，reason="携带道具"

### G5. 轴向一致性与结构暴露
- 视角轴向一致性：头部、胸腔（躯干）与骨盆的主朝向应在三视图中保持同一角色的空间逻辑连续。允许少量自然转动、重心偏移与透视差，但若出现无法由视角变化解释的朝向矛盾，例如头、躯干、骨盆彼此独立反向，或同一结构在相邻视图中发生接近/超过 90° 的突变性扭转 → pass=false，reason="轴向严重冲突"
- 足部投影一致性（一票否决）: 侧视图中，脚尖指向必须与身体侧向一致。 → pass=false, reason="足部轴向冲突"
- 站姿逻辑（兼容外八字）：允许脚尖自然外展，但双脚开角必须对称，重心居中。若双脚朝向杂乱或严重不对称 → pass=false，reason="站姿重心偏移"
- 外套与遮挡准则：允许穿着外套/风衣/披风/斗篷等角色设定服饰。手臂必须与躯干保持间隙（A-pose）；仅当服饰完全抹平腰部与胯部转折线条、导致躯干呈无结构的水桶状时 → pass=false，reason="身体结构遮挡"。披风/斗篷自然垂落不视为遮挡
- 背部可见性：允许背部装饰，但必须能通过光影或剪影辨识出脊柱走向及腰部位置。若背部被巨型披风、背包或长发完全覆盖导致结构不明 → pass=false，reason="背部细节遮挡"

### G6. 头部净空
- 耳机或头饰遮挡双耳/颈部线条 → pass=false，reason="头部净空违规"

### G7. 结构净空
- 大体积腰包遮挡腰线/胯部 → pass=false，reason="腰部净空违规"
- 高耸领口/巨大兜帽遮挡颈肩连接点 → pass=false，reason="肩颈净空违规"
- 腋下无间隙，手臂与躯干轮廓融合 → pass=false，reason="四肢净空违规"
- 单侧挂载重型道具导致视觉重心偏移 → pass=false，reason="单侧重心偏移"
- 双手必须裸露可见，佩戴手套/手部被遮挡 → pass=false，reason="手部净空违规"

### G8. 画风合规
- 角色整体渲染风格必须与项目视觉模式【{visual_mode}】一致
- 真人影视风：必须呈现真人影视级写实质感（真实皮肤纹理/毛孔/光影），严禁3D游戏CG风格化渲染、2D动漫/卡通/扁平画风 → pass=false，reason="画风不符：非真人影视风"
- 3D游戏CG风：必须呈现风格化3D游戏CG质感（介于写实与动漫之间），严禁真人照片级写实、2D动漫平涂 → pass=false，reason="画风不符：非3D CG风"
- 2D动漫风：必须呈现2D动漫/赛璐璐画风，严禁3D CG渲染、真人写实 → pass=false，reason="画风不符：非2D动漫风"
- 其他模式：画风必须与视觉模式描述匹配，严重偏离 → pass=false，reason="画风不符"
