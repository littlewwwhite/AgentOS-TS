基于输入主图，生成同一场景的多视图参考表，而不是单一大全景或单一反打镜头。

画面要求：
- 组成一张 16:9 场景设定板，包含 4 个清晰分区：主入口视角、反向视角、空间角落细节、俯视/布局感视角。
- 所有分区必须属于同一个空间，保持一致的地面纹理、墙面材质、环境光方向、时代风格和色彩体系。
- 每个视角都要展示可供分镜复用的空间信息，例如门窗、动线、家具/陈设位置、遮挡物、光源、可交互区域。
- 可以调整机位和焦段，但不要改变场景身份，不要生成另一个场景。

禁止：
- 不要只生成一张大全景。
- 不要只把主图裁切成多块。
- 不要出现人物、角色肖像、可读文字、水印、标题栏或 UI 边框。
- 不要把建筑或关键空间结构删除，除非它们明显不属于该场景。
