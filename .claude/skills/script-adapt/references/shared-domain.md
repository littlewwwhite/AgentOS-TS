# 共享领域知识

## 硬编码约束

全流水线锁定参数，各阶段共享：

- **平台**：抖音
- **形态**：竖屏微短剧
- **单集时长**：60 秒（Phase 1 可在此基础上调整）
- **目标受众**：由 Phase 1 分析确认（根据原文题材推断，用户可覆盖）
- **对白策略**：默认启用旁白/OS/五类功能标签
- **协议版本**：NTS v1.0

## 受众方法论

> 以下内容按 Phase 1 确认的受众类型条件应用。

### 女频创作方法论（目标受众 = 女频时适用）

#### 读者心理模型

- **代入视角**：读者默认代入女主视角；女主的情绪波动和每一个决策都需要充分的内心依据
- **爽感驱动**：核心循环是"压抑→爆发→奖赏"——压得越深，爆发越爽；打脸/逆袭必须有前期铺垫的委屈作为燃料
- **虐甜节奏**：甜与虐交替推进，虐是为了放大后续甜的体验；长线结构中甜虐比例从 3:7 渐变为 7:3

#### 情绪结构模板

- **全剧 HE 弧线**：甜虐交替 → 最大危机（黑暗时刻）→ 逆转爆发 → 大甜收尾
- **单集微循环**：开场悬念/委屈 → 中段对抗升级 → 结尾小爆发或新悬念钩子
- **母题累积**：关键意象/台词/动作在不同情境下反复出现，制造识别度与情感叠加

#### 角色对白分配模式

- **女主**：内心独白（OS）密度最高，承载情绪处理与决策推演；外在对白随成长弧从被动回应转向主动宣告
- **男主**：台词少但单句信息量极高（"一句封神"）；关键转折处用一句话扭转局势
- **功能角色**：闺蜜/丫鬟/属下作为信息中转，替主角说出不便直说的话；反派通过身份/权力碾压式台词制造压迫感

### 男频创作方法论（目标受众 = 男频时适用）

#### 读者心理模型

- **代入视角**：读者默认代入男主视角；男主的能力成长、等级突破、实力碾压需要充分展现
- **爽感驱动**：核心循环是"蛰伏→觉醒→碾压"——蛰伏越久，觉醒越燃；碾压必须有前期被低估/打压作为铺垫
- **升级节奏**：能力/地位阶梯式递进，每次突破带来新的层级碾压感

#### 情绪结构模板

- **全剧燃爆弧线**：隐忍蓄力 → 初露锋芒 → 强敌压境 → 极限爆发 → 登顶收尾
- **单集微循环**：开场挑衅/危机 → 中段能力展示/策略对抗 → 结尾碾压或新挑战出现
- **等级系统**：清晰的实力阶梯，每次跨越都有仪式感

#### 角色对白分配模式

- **男主**：台词简洁有力，关键时刻一句定场；内心独白围绕目标与决心
- **对手**：层层递进的挑衅与震惊，从轻蔑到恐惧的情绪转变
- **功能角色**：解说/见证者角色，负责侧面烘托男主实力；盟友角色推动能力升级线索

### 通用创作方法论（目标受众 = 全年龄时适用）

#### 读者心理模型

- **代入视角**：以事件驱动代替性别视角，角色动机普适化；观众代入的是"处境"而非特定性别角色
- **爽感驱动**：核心循环是"困境→突破→回报"——困境越真实，突破越有共鸣
- **节奏平衡**：紧张与舒缓交替，信息密度与情绪呼吸空间均匀分配

#### 情绪结构模板

- **全剧平衡弧线**：悬念建立 → 层层揭秘 → 最大危机 → 逆转解决 → 情感收尾
- **单集微循环**：开场悬念/问题 → 中段探索/对抗 → 结尾进展或新疑问
- **情感锚点**：每集至少一个让观众产生共情的关键时刻

#### 角色对白分配模式

- **主角**：行动驱动型对白，通过选择和行为展现性格
- **关键配角**：提供不同视角和价值观碰撞，丰富叙事层次
- **功能角色**：信息传递、节奏调节，适度幽默或紧张感

## 60 秒微短剧时间轴结构

每集节拍表必须对齐此时间框架：

```
0–3 秒 ：钩子（声画同步：画面抓眼球+角色直接开口，0帧起手抓注意力）
3–10 秒：建立（交代本集核心矛盾，快速入戏）
10–35秒：升级（冲突层层加码，至少2次小转折）
35–50秒：高潮（情绪最高点/最大反转/打脸爆发）
50–57秒：余韵（情绪着陆，或投放新信息铺垫下集）
57–60秒：钉子（悬念钩子，驱动下一集点击）
```
