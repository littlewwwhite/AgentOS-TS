# S2 基础设定 — 交付物模板

> 本文件定义 S2 阶段所有交付物的标准输出格式。写作时按此模板填充具体内容。

---

## 交付物 ① 世界观设定文档

```
[SWS-ITEM: 世界观设定文档]
【世界观设定文档】

一、基本信息
  项目名称：[剧名/暂定名]
  题材标签：[如"都市/重生/复仇"]
  目标受众：女频
  平台：抖音竖屏微短剧
  单集时长：60秒
  建议集数：[基于 S1 建议]
  Logline：[来自 S1 创意概念报告]

二、时空背景
  时代：[现代/古代/架空/未来]
  核心地点：[城市/国度/世界名称]
  时间跨度：[故事发生的时间范围]
  社会环境：[1-2句话概括社会氛围]

三、世界规则
  核心法则①：[该世界最重要的运行规则，1句话]
  核心法则②：[第二重要的规则，1句话]
  潜规则：[表面规则下的真实逻辑，1句话]
  （规则数量视创意复杂度而定，通常2-3条）

四、社会结构
  顶层：[权力/财富/地位的最高层]
  中层：[主角或核心角色所处的层级]
  底层：[被压迫/被忽视的层级]
  阶层流动规则：[能否跨越？如何跨越？]

五、核心冲突
  主线冲突：[来自 S1 创作锚点清单]
  冲突根源：[冲突的深层原因]
  冲突走向：[冲突如何升级和最终解决的方向]

六、主要角色（简要）
  [角色名]：[一句话定位 + 在世界观中的位置]
  [角色名]：[一句话定位 + 在世界观中的位置]
  ...
  （详细角色设定在 S4 角色开发阶段展开）

七、主要地点
  地点①：[名称] — [功能描述] — [氛围感]
  地点②：[名称] — [功能描述] — [氛围感]
  地点③：[名称] — [功能描述] — [氛围感]
  ...
  （控制在5-8个核心地点，优先考虑场景复用）

八、情节方向（简要）
  起：[开局状态]
  承：[冲突升级方向]
  转：[核心转折方向]
  合：[结局走向 HE/BE/OE]
  （详细情节在 S3 故事大纲阶段展开）

九、爽点/虐点设计方向
  核心爽点类型：[打脸/逆袭/撒糖/金手指碾压/身份揭秘...]
  核心虐点类型：[误会/背叛/牺牲/错过/生离死别...]
  爽虐节奏：[爽多虐少/虐多爽少/交替均衡]

十、备注
  [其他需要记录的设定信息]

十一、版本记录
  v1.0 [日期] — 初版，基于 S1 交付物生成
[/SWS-ITEM]
```

## 交付物 ② 核心势力/阵营设定

```
[SWS-ITEM: 核心势力/阵营设定]
【核心势力/阵营设定】

势力格局概述：[一句话概括势力关系，如"两大家族暗中角力，女主夹在其中"]

阵营①：[名称]
  定位：[一句话定位]
  核心人物：[该阵营的关键角色]
  权力基础：[权力/财富/武力/信息来源]
  对外立场：[表面立场]
  隐藏目的：[真实意图]

阵营②：[名称]
  定位：[一句话定位]
  核心人物：[该阵营的关键角色]
  权力基础：[权力/财富/武力/信息来源]
  对外立场：[表面立场]
  隐藏目的：[真实意图]

阵营③（如有）：[名称]
  定位：[一句话定位]
  核心人物：[该阵营的关键角色]
  权力基础：[权力/财富/武力/信息来源]

势力关系：
  [阵营①] ↔ [阵营②]：[对立/合作/暗中博弈] — [一句话描述]
  [阵营①] ↔ [阵营③]：[关系] — [一句话描述]
  [阵营②] ↔ [阵营③]：[关系] — [一句话描述]

主角位置：[主角在势力格局中的位置和立场变化方向]
[/SWS-ITEM]
```

## 交付物 ③ 特殊设定卡

```
[SWS-ITEM: 特殊设定卡]
【特殊设定卡】

一、金手指设定
  名称：[金手指名称/代号]
  来源：[如何获得：天生/觉醒/外物/重生自带]
  能力描述：[具体能做什么，2-3句话]
  触发条件：[何时/如何触发]
  限制条件：[使用代价/冷却/次数限制]
  能力边界：
    ✅ 能做：[明确列举]
    ❌ 不能做：[明确列举]
  升级路径（如有）：
    Lv.1 [初始能力]
    Lv.2 [成长后能力]
    Lv.3 [最终形态]
  表现形式：[金手指触发时的标志性表现，如何让观众感知]

二、特殊机制（如有）
  机制名称：[如"重生规则""契约规则""血脉觉醒"]
  规则说明：[该机制如何运作]
  对剧情的影响：[该机制如何推动或限制剧情]

三、设定一致性检查
  金手指 vs 核心冲突：[金手指是否服务于核心冲突？]
  金手指 vs 第一推动力：[金手指是情感催化剂还是事件催化剂？]
  限制条件 vs 剧情张力：[限制条件是否足以维持悬念？]
[/SWS-ITEM]
```

```
[SWS-ITEM: 风格DNA卡]
【风格 DNA 卡】

风格方向：[所选风格代号与名称，来自 S1 风格方案]

核心意象：[反复出现的视觉/情感意象]
情绪基调：[如"压抑中透希望"]
对话密度：[高/中/低]
禁区词汇：[不会出现的词汇类型]
[/SWS-ITEM]
```
