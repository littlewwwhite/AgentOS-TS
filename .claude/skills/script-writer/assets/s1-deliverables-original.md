# S1 创意构思 — 交付物模板

> 本文件定义 S1 阶段所有交付物的标准输出格式。写作时按此模板填充具体内容。

---

## 交付物 ① 创意概念报告

```
[SWS-ITEM: 创意概念报告]
【创意概念报告】

Logline：[一句话概括核心故事]
题材标签：[如"都市/重生/复仇"]
核心梗：[故事核心创意，如"重生复仇""真假千金"]
核心期待感：[观众最期待看到的内容]

六维度画像：
  Identity：[主角身份/性格关键词]
  Relationship：[核心关系模式]
  World：[世界观设定]
  Hook：[核心钩子/卖点]
  Vibe：[风格调性]
  Goldfinger：[金手指/外挂]

第一推动力：[情感推动力/剧情推动力]
推动力分析：[为什么是这个推动力，1-2句话]

背景设定：[时代、地点、社会背景]
主线冲突：[核心矛盾一句话]
核心角色构想：[角色名 + 一句话定位]（初步，S2 详细展开）
建议集数：[基于创意复杂度和节拍密度的建议]
[/SWS-ITEM]
```

## 交付物 ② 创作锚点清单

> 创作锚点 = 创意中不可更改的核心设定，是全流水线的底线约束。

```
[SWS-ITEM: 创作锚点清单]
【创作锚点清单】

第一推动力：[情感推动力/剧情推动力] — 全剧核心驱动逻辑不可更改
世界观：[核心世界规则/设定，不可更改]
主线冲突：[故事主要矛盾，不可更改]
角色核心设定：
  - [角色名]：[不可更改的核心设定，如身份/性格底色/能力]
  - ...
角色弧光：
  - [角色名]：[成长轨迹，从A到B的转变]
  - ...
关键关系锚点：[核心人物关系的起点和终点]
结局走向：[HE/BE/OE，及核心结局事件]
关键意象/道具：[贯穿全剧的标志性元素]
金手指规则：[金手指的能力边界和限制]
[/SWS-ITEM]
```

## 交付物 ③ 风格方案

```
[SWS-ITEM: 风格方案]
【风格方案】

第一推动力：[情感推动力/剧情推动力]

推荐主风格：[代号 + 名称，如"A. 虐心催泪流"]
推荐理由：[基于六维度特征和第一推动力的分析]

推荐副风格（可选）：[代号 + 名称]
组合理由：[为什么需要副风格辅助]

建议主副比例：[如 主70% / 副30%]

核心风格特征预览：
  对白特点：[该风格的对白特征]
  节奏特点：[该风格的节奏特征]
  画面调性：[该风格的画面调性]
  适用题材：[该风格适用的题材]
[/SWS-ITEM]
```

> 风格选项详见 `references/style-options.md`。
