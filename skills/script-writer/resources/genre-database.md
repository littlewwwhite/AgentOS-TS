# 网文题材类型数据库

本文件整理常见的网文题材类型及其特点，供创作参考。

---

## 玄幻/仙侠类

### 传统修仙
**特点**:升级流、境界体系、夺宝、打怪
**核心爽点**:实力提升、越级挑战、获得宝物
**代表作**:《凡人修仙传》《仙逆》

**常见设定**:
- 炼气→筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫
- 宗门、家族、散修
- 灵根、丹药、法宝、功法

### 苟道流
**特点**:主角极度谨慎、稳健发育
**核心爽点**:扮猪吃虎、深藏不露、稳健发育
**代表作**:《稳健师兄》《我的长生果》

**常见设定**:
- 主角绝不冒险
- 有挂不用，留着后手
- 所有人都以为主角是弱鸡

### 家族修仙
**特点**:以家族为单位的修仙
**核心爽点**:家族兴旺、培养后代、对抗外敌
**代表作**:《玄鉴仙族》《长生家族》

**常见设定**:
- 多代人接力修仙
- 家族族长视角
- 家族兴旺vs家族衰败

### 修仙搞笑
**特点**:幽默轻松、搞笑系统
**核心爽点**:轻松搞笑、欢乐解压
**代表作**:《从前有座灵剑山》《修真聊天群》

**常见设定**:
- 脑洞大开
- 反套路
- 现代梗融入修仙

## 都市类

### 都市神豪
**特点**:获得系统，无限财富
**核心爽点**:打脸、装逼、花钱
**代表作**:《神豪朋友圈》《神豪从签到开始》

**常见设定**:
- 系统奖励金钱/豪车/豪宅
- 前女友打脸
- 商业帝国

### 都市修仙
**特点**:都市背景下的修仙
**核心爽点**:实力碾压、装逼、打脸
**代表作**:《修真聊天群》《重生之都市修仙》

**常见设定**:
- 现代都市+修仙体系
- 隐藏的修仙界
- 灵气复苏

### 都市异能
**特点**:觉醒特殊能力
**核心爽点**:异能战斗、守护城市
**代表作**:《龙族》《全职高手》

**常见设定**:
- 异能觉醒
- 超自然事件
- 异能者组织

## 都市重生类

### 重生都市
**特点**:重生回过去，改变命运
**核心爽点**:先知优势、弥补遗憾、人生逆袭
**代表作**:《重生之官路商途》《重生之财源滚滚》

**常见设定**:
- 重生回学生时代
- 知道未来走势
- 赚钱、泡妞、改变命运

### 重生复仇
**特点**:重生复仇，惩恶扬善
**核心爽点**:复仇打脸、快意恩仇
**代表作**:《重生之香江1974》《重生之大涅槃》

## 历史/穿越类

### 架空历史
**特点**:穿越到古代，改变历史
**核心爽点**:知识碾压、建功立业、改变历史
**代表作**:《随波逐流之一代军师》《庆余年》

**常见设定**:
- 现代知识碾压古代
- 科举、打仗、经商
- 宫廷斗争

### 穿越经商
**特点**:利用现代知识经商
**核心爽点**:赚钱、 technological碾压
**代表作**:《寒门枭士》《北宋小厨师》

## 悬疑/惊悚类

### 都市灵异
**特点**:都市背景下的灵异事件
**核心爽点**:解谜、破案、驱鬼
**代表作**:《我有一座冒险屋》《神秘复苏》

**常见设定**:
- 鬼怪、灵异事件
- 主角有特殊能力
- 恐怖氛围

### 克苏鲁
**特点**:克苏鲁神话体系
**核心爽点**:探索未知、对抗古神
**代表作**:《诡秘之主》《道诡异仙》

**常见设定**:
- 古神、旧日支配者
- 疯狂值、理智值
- 诡异世界

## 科幻/未来类

### 星际文明
**特点**:星际战争、文明升级
**核心爽点**:科技碾压、征服星辰
**代表作**:《吞噬星空》《星河大帝》

**常见设定**:
- 宇宙文明等级
- 机甲、战舰
- 基因进化

### 末世求生
**特点**:末日背景下的生存
**核心爽点**:生存、升级、建立势力
**代表作**:《末世大回炉》《全球进化》

**常见设定**:
- 丧尸、变异
- 系统觉醒
- 基地建设

## 游戏竞技类

### 电竞
**特点**:游戏竞技
**核心爽点**:技术碾压、团队配合、夺冠
**代表作**:《全职高手》《联盟之谁与争锋》

**常见设定**:
- LOL、DOTA、王者荣耀
- 主角是大神
- 从低谷到巅峰

### 虚拟网游
**特点**:虚拟现实游戏
**核心爽点**:游戏升级、获得装备、PK
**代表作**:《网游之近战法师》《全职高手》

**常见设定**:
- 游戏头盔
- 职业体系
- 装备、技能
