# S7 逐层写作 — 交付物模板

> 本文件定义 S7 阶段的剧本输出格式。所有剧本必须严格遵循此格式。

---

## 剧本格式规范

```
第X集                              ← 集标题

X-Y 日/夜 内/外 场景名              ← 场次编号
人物：主角A、主角B / 配角：配角C / 龙套：路人甲；NPC：侍卫×2   ← 人物行（分层标注）

▲【标签】画面内容                   ← 画面行（标签：叙事/演绎/空镜）
角色名（情绪）：对白                ← 对白行
（X卡）                            ← 卡点分隔
```

---

## 格式要素说明

### 集标题
```
第1集
```

### 场次编号
```
X-Y 日/夜 内/外 场景名
```
- X = 集数，Y = 场次序号
- 日/夜：时间
- 内/外：空间
- 场景名：具体场景名称

### 人物行
```
人物：苏青鱼、庄晓蝶 / 配角：闻雪茶 / 龙套：邻居大妈；NPC：路人×3
```
- 用 `/` 分隔三个层级
- 在场必标注（即使不说话）
- 群众用"若干"，跟班用"×数量"

### 画面行
```
▲【叙事】两人隔着小圆桌对坐，苏瑶不停搅拌咖啡，勺子碰杯壁叮叮作响
▲【演绎】楚凡侧身闪过直拳，反手扣住对手手腕 → 膝盖顶上腹部
▲【空镜】倾盆大雨 → 路灯孤影 → 水洼倒映破碎天空
```
- 标签三选一：叙事（对白驱动）/ 演绎（动作驱动）/ 空镜（无角色）
- 动作点用 `→` 衔接，每行 ≤ 5 个动作点
- 每个动作点 = 3 秒连贯可见动作

### 对白行
```
角色名（情绪）：台词
角色名（OS）：内心独白
角色名（VO）：画外音
【广播】：系统/广播内容
台词[心声]隐秘心思
```

### 视觉叠层
```
【字幕：F级异能【按摩】】
【系统提示：检测到S级异能失控。】
```

### 卡点分隔
```
（1卡）
```

## 逐场自检行模板

> 每场完成后在场次末尾附加（最终交付可选保留或删除）：

```
<!-- 自检 X-Y：
  节奏轨：开场钩子[✅/❌] | 转折×[N] | 峰[场次] 谷[场次] | 钉子[✅/❌]
  画面轨：动作点饱和[N/N✅] | 标签比[叙X/演Y/空Z]
  对白轨：功能标签[命中类型] | 最长对白[N字/容量M字✅]

-->
```
