# S3 故事大纲 — 交付物模板

> 本文件定义 S3 阶段所有交付物的标准输出格式。写作时按此模板填充具体内容。

---

## 交付物 ① 故事核心

```
[SWS-ITEM: 故事核心]
【故事核心】

Logline：[一句话概括全剧]
核心冲突：[主角 vs 对立力量，冲突本质]
主题句：[全剧精神内核一句话]
第一推动力：[情感推动力/剧情推动力]

表层主题：[表面上讲什么]
深层主题：[实际上想表达什么]
[/SWS-ITEM]
```

## 交付物 ② 总体结构图

```
[SWS-ITEM: 总体结构图]
【总体结构图】

总集数：[N]
单集时长：60秒
四阶段分配：起[X集] / 承[X集] / 转[X集] / 合[X集]

起阶段（第1-X集，20%）：
  阶段目标：[叙事任务]
  情绪基调：[基调描述]
  阶段高潮：第X集 — [事件描述]
  阶段成果：[主角/关系/局势变化]

承阶段（第X-X集，30%）：
  阶段目标：[叙事任务]
  情绪基调：[基调描述]
  阶段高潮：第X集 — [事件描述]
  阶段成果：[主角/关系/局势变化]

转阶段（第X-X集，30%）：
  阶段目标：[叙事任务]
  情绪基调：[基调描述]
  阶段高潮：第X集 — [事件描述]
  阶段成果：[主角/关系/局势变化]

合阶段（第X-X集，20%）：
  阶段目标：[叙事任务]
  情绪基调：[基调描述]
  终极高潮：第X集 — [事件描述]
  结局收束：[主角/关系/局势最终状态]
[/SWS-ITEM]
```

## 交付物 ③ 分集大纲

```
[SWS-ITEM: 分集大纲]
【分集大纲】

第1集：[Logline] | 核心事件：[事件] | 情绪：[起→峰→谷→终] | 转折×[N>=4] | 钉子：[集尾悬念]
第2集：[Logline] | 核心事件：[事件] | 情绪：[起→峰→谷→终] | 转折×[N>=4] | 钉子：[集尾悬念]
...
第N集：[Logline] | 核心事件：[事件] | 情绪：[起→峰→谷→终] | 转折×[N>=4] | 钉子：[集尾悬念]
[/SWS-ITEM]
```

## 交付物 ⑥ 爽虐分布图

```
[SWS-ITEM: 爽虐分布图]
【爽虐分布图】

阶段 | 集数 | 爽/虐 | 类型 | 程度 | 事件描述
起   | 第1集  | 爽   | [类型] | 小爽 | [描述]
起   | 第2集  | 虐   | [类型] | 小虐 | [描述]
...
承   | 第X集  | 爽   | [类型] | 中爽 | [描述]
...
转   | 第X集  | 虐   | [类型] | 大虐 | [描述]
...
合   | 第X集  | 爽   | [类型] | 超大爽 | [描述]

情绪曲线总览：
  起阶段：爽 > 虐（吸引代入）
  承阶段：爽 ≈ 虐（交替推进）
  转阶段：虐 > 爽（危机断裂）
  合阶段：大爽收尾（情绪释放）
[/SWS-ITEM]
```

## 交付物 ⑦ 卡点预设方案

```
[SWS-ITEM: 卡点预设方案]
【卡点预设方案】

卡点位置：第X集尾
卡点悬念类型：[身份揭晓/生死悬念/关系断裂/真相临门/选择困境]
卡点悬念内容：[具体悬念描述]

情绪加码链（卡点前 3-5 集）：
  第X集：[加码事件] → 情绪推向 [方向]
  第X集：[加码事件] → 情绪推向 [方向]
  第X集：[加码事件] → 情绪推向 [方向]
  第X集（卡点集）：[最强悬念落下] → 观众情绪到达"不看下去不行"的临界点

付费首集设计（第X集）：
  开场钩子：[立即回应卡点悬念的部分内容]
  新悬念投放：[给出部分满足的同时抛出新问题]
[/SWS-ITEM]
```
