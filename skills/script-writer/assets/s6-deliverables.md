# S6 单集细纲 — 交付物模板

> 本文件定义 S6 阶段所有交付物的标准输出格式。S6 产出是 S7 的直接输入，格式必须严格匹配。

---

## 交付物 ① 结构总量规格

```
[SWS-ITEM: 结构总量规格]
┌──────────────────────────────────┐
│ 结构总量规格                      │
│ 总集数：[N]                      │
│ 单集时长：[60s]                  │
│ 总时长：[N×60s]                  │
│ 总场景数：[M]                    │
│ 卡点设置：[描述]                 │
└──────────────────────────────────┘
[/SWS-ITEM]
```

## 交付物 ② 全集节拍表（每集一张）

> ⚠️ 表头字段（时间段、节拍、场次、事件、功能、转折标记）不可增删改，S7 依赖此格式。

```
[SWS-ITEM: 全集节拍表]
【第X集节拍表】

集 Logline：[一句话概括]
集目标：[叙事目标]
情绪曲线：[起点] → [↑峰：XX段] → [↓谷：XX段] → [终点]
开场钩子类型：[画面：冲突/反差/危机/极端表情] + [台词：悬疑/欲望/矛盾/反转/情绪炸弹]

| 时间段 | 节拍 | 场次 | 事件 | 功能 | 转折标记 |
|--------|------|------|------|------|---------|
| 0–3秒  | 钩子 | X-1  | [事件] | 0帧起手：声画同步开炸 | 🔄 T1 |
| 3–15秒 | 建立 | X-1  | [事件] | 交代矛盾/建立冲突 | |
| 15–30秒| 升级 | X-2  | [事件] | 冲突加码/信息投放 | 🔄 T2 |
| 30–45秒| 高潮 | X-3  | [事件] | 情绪顶点/核心冲突 | 🔄 T3 |
| 45–55秒| 余韵 | X-3  | [事件] | 情绪着陆/决策落地 | 🔄 T4 |
| 55–60秒| 钉子 | X-3  | [事件] | 悬念钩子→衔接下集 | |

集尾悬念：[悬念内容]
→ 下集衔接：[与下集开场钩子的关系：因果/对照/回答]

（全部集数的节拍表）
[/SWS-ITEM]
```

## 交付物 ③ 场景清单

> ⚠️ 行格式（场次编号 + 日/夜 + 内/外 + 场景名 + 角色分层标注 + 目标）不可变动，S7 依赖此格式。

```
[SWS-ITEM: 场景清单]
【场景清单】

第1集：
  1-1 [日/夜] [内/外] [场景名] | 主角：[角色名] | 配角：[角色名] | 龙套/NPC：[描述性标注] | 目标：[一句话]
  1-2 ...
第2集：
  2-1 ...
...

总场景数：[N]
[/SWS-ITEM]
```

## 交付物 ④ 关键集数细纲（可选但推荐）

> 建议对第1集、卡点集、大高潮集、大结局集进行更详细规划。

```
[SWS-ITEM: 关键集数细纲]
【第X集细纲】（关键集标注：开篇/卡点/大高潮/大结局）

一、本集定位
- 阶段位置：[开端/发展/高潮/结局]
- 承接上集：[上集钉子如何衔接到本集]
- 引向后集：[本集为后续铺垫了什么]

二、核心情节
- 主要事件：[事件描述]
- 状态推进：[开始状态] → [发展变化] → [结束状态]

三、出场角色
| 角色 | 层级 | 本集行动 | 关系变化 |
|------|------|----------|----------|
| [角色名] | 主角 | [做了什么] | [关系如何变化] |
| [角色名] | 配角 | [做了什么] | [关系如何变化] |

四、爽虐点安排
- 爽点：[类型] | 时间点：[XX秒] | 强度：[小爽/中爽/大爽]
- 虐点：[类型] | 时间点：[XX秒] | 强度：[小虐/中虐/大虐]

五、伏笔处理
- 埋下：[伏笔内容] → 计划回收：第X集
- 回收：[回应第X集埋下的XX伏笔]

六、对话重点
- 关键对话：[角色A]与[角色B]的[对话方向]
- 金句方向：[设计意图]
- 语言特征：[需要体现的角色语言风格]

（对每个关键集数重复以上结构）
[/SWS-ITEM]
```
