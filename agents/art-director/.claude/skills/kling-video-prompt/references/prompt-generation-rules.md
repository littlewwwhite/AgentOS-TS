# 完整提示词生成规则

## 常见错误模式

提示词生成中最常见的问题是**只包含剧本原文**，缺少关键的视觉描述元素。

### 错误示例
```
保持人物与参考图完全一致... 角色A凑到角色B耳边，声音压得很低 → 眼神警惕地瞥向角色C 角色A（提醒）："对话内容"
```

### 正确示例
```
保持人物与参考图完全一致... 0-3s，85mm中长焦中近景镜头缓缓推入，浅景深。【场景名】内景，午后暖光透过丝绸帷幕洒入，柔和金色光晕。【角色A】凑近【角色B】耳边，面容紧张而急切，声音刻意压低成耳语，眼神不安地瞥向背景中的【角色C】。【角色B】表情平静但眼中闪过一丝算计...
```

## 完整提示词必须包含的元素

### 1. 人物一致性前缀（必须）
```
英文：Maintain characters exactly as reference images, 100% identical facial features, same bone structure, eye spacing and jaw geometry, no beautification, no age changes.

中文：保持人物与参考图完全一致，面部特征100%相同，保持相同的骨骼结构、眼距和下颚几何形状，禁止美化，禁止改变年龄。
```

### 2. 时间标记（必须）
- 格式：`0-3s, 3-7s, 7-10s`
- 每个shot对应一个时间段
- 时间从0开始，连续不间断

### 3. 镜头语言（必须）
包含以下元素：

**a) 景别（Shot Size）**
- 大远景 Extreme Wide Shot
- 远景 Long Shot
- 全景 Full Shot
- 中景 Medium Shot
- 中近景 Medium Close-Up
- 近景 Close Shot
- 特写 Close-Up
- 大特写 Extreme Close-Up

**b) 运镜（Camera Movement）**
- 推镜 Dolly In / Push In
- 拉镜 Pull Back / Dolly Out
- 摇镜 Pan
- 跟拍 Follow Shot
- 环绕 Orbit
- 升降 Crane
- 手持 Handheld
- 稳定器 Gimbal

**c) 焦距（Focal Length）**
- 广角 24-35mm
- 标准 40-60mm
- 中长焦 70-105mm
- 长焦 135-200mm

**d) 景深（Depth of Field）**
- 浅景深 Shallow depth of field
- 深景深 Deep depth of field
- 背景虚化 Background bokeh

**e) 构图（Composition）**
- 三分法 Rule of Thirds
- 中心构图 Center Composition
- 对角线构图 Diagonal Composition
- 引导线 Leading Lines
- 过肩 Over-the-Shoulder

### 4. 光影描述（必须）
- 光源方向：侧光、顶光、逆光、正面光
- 光线质量：柔和、刺眼、温暖、冷峻
- 明暗对比：高调、低调、强对比
- 氛围：金色光晕、蓝色冷光、暖黄色调

示例：
- "午后暖光透过丝绸帷幕洒入，柔和金色光晕"
- "刺眼的午后阳光从窗户射入，形成强烈明暗对比"

### 5. 场景描述（必须）
- 内/外景标注
- 场景名称用【】标注
- 环境细节：装饰、家具、氛围

示例：
- "【灵霜寝宫】内景，华丽的红木家具，丝绸帷幕"

### 6. 人物描述（必须）
**a) 位置和姿态**
- 在场景中的位置
- 身体姿态
- 与其他角色的空间关系

**b) 表情和眼神**
- 面部表情的细节
- 眼神的方向和情绪
- 微表情的变化

**c) 动作细节**
- 具体的动作描述
- 动作的速度和力度
- 肢体语言

**d) 【】标注**
- 所有人物名用【】标注
- 示例：【陆云】【灵霜】【白行风】

### 7. 道具描述（如有）
- 道具名称用【】标注
- 道具的状态和使用方式
- 示例：【轮椅】【内丹】【剑】

### 8. 对话（如有）
- 格式：`角色名（情绪）："对话内容"`
- 保留中文原文
- 标注语气和情绪

示例：
- `陆云（提醒语气）："师姐，你可千万不要修复师兄的丹田啊。"`

### 9. 内心独白处理（如有）
**重要**：不直接写出内心独白文字，而是通过视觉细节暗示

错误做法：
```
白行风想："百年的婚约和相伴……"
```

正确做法：
```
【白行风】眼神空洞，眼角缓缓凝聚出一滴泪水。他的目光扫过周围堆满的奢华礼物——都是他送给【灵霜】的，此刻看来像是残酷的嘲讽。
```

## 生成流程

### 步骤1：分析剧本内容
- 提取动作描述
- 提取对话/内心独白
- 理解情绪和氛围

### 步骤2：设计镜头语言
- 根据情绪选择景别（亲密→近景，宏大→远景）
- 根据节奏选择运镜（紧张→快速，舒缓→缓慢）
- 根据重点选择焦距（人物→长焦，环境→广角）

### 步骤3：扩展视觉细节
- 添加环境描述
- 细化人物表情和动作
- 添加光影氛围

### 步骤4：添加【】标注
- 标注所有人物
- 标注场景
- 标注道具

### 步骤5：整合时间线
- 按shot分段
- 添加时间标记
- 确保连贯性

保持人物与参考图完全一致... 陆云凑到灵霜耳边，声音压得很低 → 眼神警惕地瞥向白行风 陆云（提醒）：师姐，你可千万不要修复师兄的丹田啊。

### 完整版（合格）
```
保持人物与参考图完全一致，面部特征100%相同，保持相同的骨骼结构、眼距和下颚几何形状，禁止美化，禁止改变年龄。0-3s，85mm中长焦中近景镜头缓缓推入，浅景深。【灵霜寝宫】内景，午后暖光透过丝绸帷幕洒入，柔和金色光晕。【陆云】凑近【灵霜】耳边，面容紧张而急切，声音刻意压低成耳语，眼神不安地瞥向背景中的【白行风】。【灵霜】表情平静但眼中闪过一丝算计。虚化的背景中，【白行风】坐在【轮椅】上一动不动，似乎毫无察觉。陆云（提醒语气）："师姐，你可千万不要修复师兄的丹田啊。" 3-7s，从【灵霜】视角的过肩镜头，标准镜头，中景深。【陆云】的脸部充满画面，眼神强烈而恳求，眉头微蹙。他的手微妙地朝【白行风】方向示意。暖光照亮他额头上的忧虑纹路。背景显示寝宫华丽内景，屏风和丝绸帷幕柔焦虚化。
```
