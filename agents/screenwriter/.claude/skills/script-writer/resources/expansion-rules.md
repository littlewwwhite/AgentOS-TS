# NTSV2 扩写规则 v1.0

> **扩写规则**定义了将短篇小说（约1万字）扩展为50-60集微短剧剧本的方法论，是 NTSV2 流水线中从原文到剧本的核心转化指南。

---

## 一、扩写总则

### 1.1 核心原则：保核扩外

扩写的本质是**保持原文核心不变，扩展外围内容**。原文是种子，扩写是让种子长成大树——根系（核心）不变，枝叶（外围）自由生长。

### 1.2 扩写比例指导

- **输入**：原文约1万字短篇小说
- **输出**：50-60集微短剧剧本（每集约60秒，总时长约60分钟）
- **扩写倍率**：内容量扩展5-8倍（按剧本信息密度计算）

### 1.3 扩写的5不变

无论如何扩写，以下5个要素**绝对不可更改**：

| 序号 | 不变项 | 说明 |
|------|--------|------|
| 1 | **主线冲突不变** | 原文的核心矛盾/对立关系必须保留，不可替换为新冲突 |
| 2 | **角色核心设定不变** | 角色的本质性格、核心身份、关键能力不可更改 |
| 3 | **结局走向不变** | 原文的结局方向（HE/BE/OE）和关键结果不可篡改 |
| 4 | **IP不动点不变** | 原文最具辨识度的标志性元素（金句、名场面、独特设定）必须保留 |
| 5 | **第一推动力不变** | 驱动整个故事启动的核心事件/动机不可替换 |

## 二、六大扩写维度

### 2.1 主线细化

**目标**：将原文概述式事件拆解为可演的场景序列。

**方法——事件拆解法**：

```
一个大事件 → 3-5个子事件 → 每个子事件 = 1-2集
```

**操作步骤**：

1. 提取原文主线事件列表（通常5-10个关键节点）
2. 每个关键节点拆解为"起因-经过-结果"三幕
3. 每个三幕结构进一步拆解为具体的可演场景
4. 在相邻事件之间插入过渡事件（铺垫、伏笔、呼应）

**约束**：
- 主线事件的**顺序不可更改**
- 可以在主线事件之间**插入过渡事件**，但过渡事件不得喧宾夺主
- 每个主线事件的**核心结果不可更改**（过程可丰富）

**示例**：

```
原文："女主发现男主的秘密身份后决定离开"（一句话）
↓ 拆解为：
子事件1：女主无意中发现疑点（1集）
子事件2：女主暗中调查，逐步接近真相（1-2集）
子事件3：真相揭露的名场面（1集）
子事件4：女主的内心挣扎与决定（1集）
子事件5：离开的执行与男主的反应（1集）
= 共5-6集
```

### 2.2 支线补写

**目标**：创造新的支线剧情，丰富故事层次。

**支线来源**：

| 来源类型 | 说明 | 示例 |
|---------|------|------|
| 配角独立故事线 | 为原文配角补充独立的故事弧 | 闺蜜的感情线、反派的前史 |
| 原文暗示未展开 | 原文提及但未详述的情节展开 | "多年前的一场意外"→完整回溯 |
| 世界观衍生冲突 | 从世界设定中派生新的矛盾 | 家族企业内部的权力斗争 |

**约束**：
- 支线**必须与主线产生交互**，交互方式包括：推动主线、阻碍主线、揭露信息、映衬主题
- 50集剧建议 **3-5条支线**
- 每条支线至少在**3个以上集数**中出现
- 支线不得独立于主线运行超过**连续3集**

**支线与主线的交互矩阵**：

```
支线A（配角感情线）
  ├─ 第12集：配角无意间获取关键信息 → 推动主线
  ├─ 第25集：配角立场动摇 → 阻碍主线
  └─ 第40集：配角的选择映衬主题

支线B（反派前史线）
  ├─ 第8集：闪回揭示反派动机 → 揭露信息
  ├─ 第30集：反派过去的创伤影响当下决策 → 推动主线
  └─ 第50集：反派的结局映衬主角成长 → 映衬主题
```

### 2.3 角色深化

**目标**：为原文中扁平的角色增加深度，使其成为立体的人物。

**深化方法**：

| 方法 | 说明 | 占用集数 |
|------|------|---------|
| 补充背景故事 | 通过闪回或对话揭示角色过去 | 穿插于主线中 |
| 增加内心戏 | 通过独白、表情、行为细节展现内心 | 不占独立集数 |
| 设计成长弧光 | 角色从A状态到B状态的转变过程 | 贯穿全剧 |
| 丰富人物关系 | 增加角色之间的互动维度 | 穿插于主线中 |

**约束**：
- **不可违背**原文已确立的角色核心性格
- 新增背景故事**不可与原文矛盾**
- 角色深化内容必须**服务于剧情发展**，不可为深化而深化

**角色深化检查表**：

```
□ 该角色是否有明确的"想要"（Want）和"需要"（Need）？
□ 该角色是否有独特的说话方式和行为习惯？
□ 该角色是否有足以解释其行为的背景故事？
□ 该角色在故事结束时是否与开始时有所不同？
□ 该角色与主角的关系是否有变化和发展？
```

### 2.4 世界观补足

**目标**：扩展原文的世界设定，使故事背景更加完整可信。

**方法**：

从原文暗示的规则/环境出发，补充以下维度：

1. **社会结构**：阶层、组织、权力关系
2. **规则体系**：世界运行的基本法则（现实向为社会规则，幻想向为超自然法则）
3. **地理环境**：故事发生的空间格局
4. **文化背景**：影响角色行为的文化因素
5. **历史脉络**：影响当前局势的历史事件

**约束**：
- **不可与原文世界观矛盾**
- 世界观补足必须**为剧情服务**，不可进行无关的设定堆砌
- 世界观信息通过**剧情自然展现**，避免大段说明性台词

### 2.5 情感扩写

**目标**：拉长和丰富情感线，增加观众的情感投入。

**方法**：

| 手法 | 说明 | 适用阶段 |
|------|------|---------|
| 增加暧昧期 | 延长角色之间互相试探的过程 | 感情线前期 |
| 增加误会环节 | 因信息不对称产生的情感波折 | 感情线中期 |
| 增加分离与重逢 | 物理或心理上的分离后再相聚 | 感情线中后期 |
| 增加情感考验 | 外部压力对情感的考验 | 感情线高潮期 |
| 增加甜蜜互动 | 日常互动中展现感情升温 | 穿插全程 |

**甜虐节奏**：

```
每5-8集形成一个甜虐小循环：

甜（1-2集）→ 虐（2-3集）→ 更甜（1-2集）→ 更虐（2-3集）
                                                    ↓
                                              进入下一循环
```

**整体情感节奏参考**（以55集为例）：

```
第01-10集：相识 → 暧昧萌芽（甜7虐3）
第11-20集：感情升温 → 第一次危机（甜5虐5）
第21-30集：短暂甜蜜 → 重大误会/分离（甜3虐7）
第31-40集：各自成长 → 重逢 → 考验（甜4虐6）
第41-50集：终极考验 → 和解（甜3虐7→转甜）
第51-55集：收束 → 大结局（甜8虐2）
```

**约束**：
- 情感发展的**终点不变**，但中间过程可以更曲折
- 误会环节**不可过于低智**（角色智商需在线）
- 每段虐的内容**必须有叙事价值**（推动角色成长或揭示真相）

### 2.6 冲突升级

**目标**：在原文冲突基础上增加对抗层次，持续维持戏剧张力。

**方法**：

在原文冲突基础上，按以下阶梯逐步升级：

```
试探（1-10集）
  ↓ 双方初次交锋，互相试探底线
对撞（11-20集）
  ↓ 正面冲突爆发，双方各有损失
反扑（21-35集）
  ↓ 暂时落败方反扑，局势反转
摊牌（36-48集）
  ↓ 核心秘密揭露，所有矛盾摆上台面
终极对决（49-55集）
  → 最终决战/抉择，主线冲突彻底解决
```

**新增冲突的类型**：

| 冲突类型 | 说明 | 与主线关系 |
|---------|------|-----------|
| 新对手 | 在原有对手之外增加新的对立者 | 可作为主线对手的盟友/竞争者 |
| 新障碍 | 增加角色达成目标的难度 | 增加主角的行动成本 |
| 新危机 | 增加突发事件/不可控因素 | 迫使主角调整策略 |
| 内部冲突 | 角色阵营内部出现分歧 | 考验人物关系的牢固度 |

**约束**：
- 新增冲突**必须服务于主线冲突的升级**，不可制造无关冲突
- 冲突升级必须**合乎逻辑**，有因有果
- 每次升级后角色必须**付出代价或获得成长**

## 三、扩写质量红线

以下为扩写过程中的**绝对禁区**，触犯任何一条即需返工修改：

| 红线编号 | 红线规则 | 判定标准 |
|---------|---------|---------|
| R1 | **不可为扩写而注水** | 每集必须有信息增量（新信息/新转折/新情感/新关系变化），纯过渡集不得连续出现超过1集 |
| R2 | **不可打破因果链** | 扩写内容必须与原文因果自洽，新增事件不可产生逻辑漏洞 |
| R3 | **不可弱化原文精华** | 原文的精彩桥段（名场面、金句、高能转折）必须保留并强化呈现 |
| R4 | **不可引入无法回收的新元素** | 新增的人物/设定/伏笔必须在剧内得到回收和交代 |
| R5 | **支线数量不超过主线承载力** | 支线总量不得使主线节奏被稀释至观众失去追踪兴趣 |

**红线自检方法**：

- **R1检测**：逐集列出该集的"信息增量清单"，若无法列出至少1条有效增量，该集需重写
- **R2检测**：画出事件因果图，检查每个新增事件是否有"因"且不与原文"果"矛盾
- **R3检测**：对照原文标记的精华场景清单，确认每个精华场景在剧本中的对应位置和强化方式
- **R4检测**：建立新增元素追踪表，确认每个新元素都有回收计划
- **R5检测**：统计主线和支线的集数占比，若主线占比低于40%则需调整

## 四、扩写比例参考

### 4.1 按原文规模的扩写策略

| 原文规模 | 建议集数 | 扩写倍率 | 主线占比 | 支线占比 | 情感线占比 | 扩写重点 |
|---------|---------|---------|---------|---------|----------|---------|
| 3000-5000字 | 50-55集 | 高扩写 | 40% | 30% | 30% | 大量补写支线和情感线，世界观需从零构建 |
| 5000-8000字 | 50-60集 | 中扩写 | 50% | 25% | 25% | 均衡扩展，主线细化为主，适量支线 |
| 8000-10000字 | 55-60集 | 低扩写 | 60% | 20% | 20% | 主线细化为主，少量支线，重点在场景化呈现 |

### 4.2 各维度扩写的集数分配参考（以55集为例）

```
主线细化：约28集内容（可与支线/情感线重叠）
支线补写：约12集内容（其中约8集与主线交叉）
角色深化：穿插于各集中，不独立占集
世界观补足：约3-5集集中展现 + 穿插于各集
情感扩写：约15集内容（其中约10集与主线交叉）
冲突升级：穿插于各集中，约5集为纯冲突升级集
```

> 注意：以上数字存在重叠，因为一集中通常同时包含多个维度的内容。

### 4.3 集数分布的节奏参考

| 阶段 | 集数范围 | 主要任务 | 节奏 |
|------|---------|---------|------|
| 开篇钩子 | 第1-3集 | 建立世界观、引入主角、抛出核心悬念 | 快节奏，高密度 |
| 前期铺垫 | 第4-12集 | 角色关系建立、支线引入、第一推动力发动 | 中等节奏 |
| 中前期发展 | 第13-25集 | 主线推进、支线展开、情感升温 | 中等节奏，甜虐交替 |
| 中期高潮 | 第26-35集 | 中期大反转、关系重组、冲突升级 | 紧凑节奏 |
| 中后期深化 | 第36-45集 | 真相逼近、终极考验、角色蜕变 | 紧凑节奏，虐多于甜 |
| 高潮收束 | 第46-52集 | 终极对决、核心冲突解决 | 最快节奏 |
| 大结局 | 第53-55集 | 关系收束、支线回收、情感落点 | 舒缓节奏 |

## 五、扩写执行检查清单

### 5.1 扩写前检查（Pre-Expansion Check）

```
□ 已完成原文分析，提取了所有核心要素
□ 已确定5不变清单并锁定
□ 已识别原文精华场景并标记
□ 已确定扩写倍率和总集数目标
□ 已确定主线/支线/情感线的比例分配
□ 已规划支线数量和交互方式
```

### 5.2 逐集扩写检查（Per-Episode Check）

每集扩写完成后，必须自检以下项目：

```
□ 本集是否有至少1条信息增量？
□ 本集内容是否与原文因果自洽？
□ 本集是否推进了至少一条故事线（主线/支线/情感线）？
□ 本集是否有明确的钩子（集末悬念/情感钩子/信息钩子）？
□ 本集新增内容是否可被后续集数回收？
□ 本集是否符合当前阶段的节奏要求？
□ 本集中的角色行为是否符合其已确立的性格？
□ 本集是否触犯了任何扩写质量红线？
```

### 5.3 阶段性检查（Phase Check，每10集一次）

```
□ 过去10集中主线是否有实质性推进？
□ 每条支线是否在过去10集中至少出现1次？
□ 情感线的甜虐节奏是否符合规划？
□ 冲突层级是否按计划升级？
□ 是否出现了连续2集以上的注水内容？
□ 新增元素是否都有回收计划？
□ 观众视角是否仍能清晰追踪主线？
```

### 5.4 全剧完成检查（Final Check）

```
□ 5不变项目是否全部保持？
□ 原文精华场景是否全部保留并强化？
□ 所有支线是否全部回收？
□ 所有新增伏笔是否全部回收？
□ 角色弧光是否完整？
□ 情感线终点是否与原文一致？
□ 冲突升级阶梯是否完整走完？
□ 总集数是否在目标范围内？
□ 没有悬而未决的剧情线？
□ 大结局是否令人满足且符合原文走向？
```

## 六、扩写工作流与NTSV2流水线对接

| NTSV2阶段 | 扩写维度的主要工作 |
|-----------|------------------|
| S1-原文分析与灵感提取 | 确定5不变、识别原文精华、确定扩写倍率 |
| S2-基础设定 | 世界观设定、势力设定、特殊设定卡、风格DNA卡 |
| S3-故事大纲 | 主线细化、支线补写框架、冲突升级阶梯 |
| S4-角色开发 | 角色深化、人物关系扩展 |
| S5-进度追踪 | 扩写比例监控、节奏校验 |
| S6-单集细纲 | 逐集扩写检查、情感扩写落地 |
| S7-逐层写作 | 场景级扩写执行、每集信息增量校验 |
| S8-润色终审 | 全剧完成检查、红线终审 |
| S9-格式交付 | 最终交付 |
